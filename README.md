🌟 The Story Behind This Project

This project was born from frustration, isolation, and a dream.

Like many of you, I struggled with:

· A low-end phone that couldn't run Minecraft properly
· Limited storage space (always fighting with "storage full" messages)
The result?
A Minecraft modification that went from 330MB to just 200MB, running at 40-50 FPS on the weakest processors.
🎯 What Is Minecraft Hyper Lite?

Minecraft Hyper Lite is an extreme optimization modification of Minecraft Patched (the best optimized version of Minecraft Bedrock).

I stripped away everything unnecessary while keeping the complete gameplay experience intact.
📱 Who Is This For?

✅ Low-end phone users (MediaTek, Snapdragon 400/600 series)
✅ People with limited storage (16GB/32GB phones)
✅ Students with basic devices
✅ PvP players who need maximum FPS
✅ Anyone who wants a smooth Minecraft experience
✅ Modders who want a lightweight base for their own modifications
✅ Gamers in developing countries with older devices
🔧 What Was Removed? (The Optimization Details)

🗑️ Completely Removed:

· 📁 Beta folder (unused content, testing files)
· 🌍 All extra languages (kept ONLY English - saves ~15MB)
· 🏪 Marketplace images and assets (most players don't use it)
· 📚 All tutorials, help sections, and tips (no one reads them anyway)
· ✒️ Smooth font (saves ~1MB, default font works fine)
· 🔊 80% of mob sounds (kept only essential sounds for gameplay)
· 🎨 Most mob skins/textures (mobs appear black/purple - fully functional!)
· 🖼️ Unnecessary UI elements and images

🎨 Optimized/Reduced:

· 🖼️ Startup images quality (reduced without noticeable visual loss)
· 📦 APK structure (optimized compression)
· 🔄 File organization (better loading speed)

🎮 What Remains COMPLETELY INTACT:
· ✅ All game mechanics (crafting, mining, fighting, building)
· ✅ Multiplayer support
· ✅ Core Minecraft experience
⚡ Performance Tests (Real Results)

Test Device: MediaTek Helio G25 (One of the weakest processors available)

Test Result
Average FPS 40-50 FPS (stable)
Loading time 25-20 seconds
RAM usage 300-400 MB
Battery consumption Reduced by ~30%
Note: Minor stuttering may occur occasionally due to the game engine itself. This is beyond any modification's control.
